Current controller / gamepad templates:
- Xbox
- PS5
- PS5 (Black)
- PS5 (Cosmic Red)



Manual .py script run requires:

pip install flask flask-socketio pygame eventlet requests

Then you can run it by:

python CryptoPad.py


To build the exe yourself use:

pyinstaller CryptoPad.spec
